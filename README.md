<h1>👻 Ghost Eye Tracker</h1>
A fun web animation where a ghost follows your mouse with its eyes! Built using HTML and CSS.

<h1>🛠️ Features</h1>
<ul>
  <li>👀 Eye Tracking: The ghost's eyes follow the mouse cursor.</li>
  <li>🌐 Pure HTML & CSS: No JavaScript required!</li>
  <li>🎨 Smooth Animation: Works seamlessly across modern browsers.</li>
</ul>

<h1>📂 Project Structure</h1>
📁 ghost-eye-tracker  <br>
 ├── 📄 index.html  # Main HTML file <br>
 ├── 🎨 styles.css  # CSS animations and styles  <br>
 ├── 📄 README.md   # Project documentation  <br>


<h1>🚀 Usage </h1>
<ol>
  <li>Clone the repository: </br>
  git clone  <br>
  https://github.com/kidKrix/ghost-eye-tracker.git </br>
cd ghost-eye-tracker
</li>
